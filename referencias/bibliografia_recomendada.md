# 📚 Bibliografia Recomendada sobre LLMs, GPTs e Engenharia de Prompt

## 1. Prompt Engineering for LLMs

**Autores:** John Berryman e Albert Ziegler

### 🎯 Por que ler?

* Ensina Engenharia de Prompt de forma profissional.
* Apresenta padrões de prompting utilizados em ambientes de produção.
* Excelente para estudantes e profissionais que desejam criar GPTs personalizados.
* Possui exemplos práticos utilizando GPT, Claude e outros Large Language Models (LLMs).
* Aborda técnicas avançadas para melhorar a qualidade das respostas dos modelos.

### 🔗 Link de compra

📖 Amazon Brasil – Prompt Engineering for LLMs

https://www.amazon.com.br/dp/1098156153

---

## 2. Hands-On Large Language Models

**Autores:** Jay Alammar e Maarten Grootendorst

### 🎯 Por que ler?

* Considerado um dos livros mais completos sobre Large Language Models.
* Explica os fundamentos dos Transformers.
* Apresenta conceitos de Embeddings, RAG (Retrieval-Augmented Generation) e Fine-Tuning.
* Demonstra como construir sistemas de busca semântica.
* Ideal para estudantes de Ciência da Computação, Engenharia de Software e Ciência de Dados.
* Possui exemplos práticos utilizando Python e ferramentas modernas de IA.

### 🔗 Link de compra

📖 Amazon Brasil – Hands-On Large Language Models

https://www.amazon.com.br/dp/1098150961

Após a leitura desses livros, o estudante estará apto a:

* Criar GPTs personalizados;
